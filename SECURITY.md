# Security

If you believe you’ve found a security vulnerability, please report it responsibly.

**Do not** open a public GitHub issue for security-sensitive bugs.

- Prefer contacting the maintainers directly (e.g. via the repository owner’s contact or a private channel you already use).
- Include a clear description, steps to reproduce, and impact if possible.
- We’ll respond as we can and work with you on a fix and disclosure.

Thank you for helping keep this project and its users safe.
